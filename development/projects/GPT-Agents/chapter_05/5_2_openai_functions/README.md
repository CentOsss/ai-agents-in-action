# Лабораторная работа: Система рекомендаций с использованием функций OpenAI

## Цель работы
Создать и протестировать систему рекомендаций, использующую функции OpenAI для обработки запросов пользователей и предоставления персонализированных рекомендаций.

## Предварительные требования
1. Установленные зависимости из файла `requirements.txt`
2. Настроенный API ключ OpenAI в файле `.env`

## Структура проекта
```
5_2_openai_functions/
├── README.md
├── recommendation_system.py
└── requirements.txt
```

## Задание 1: Базовая система рекомендаций
1. Изучите код в файле `recommendation_system.py`
2. Запустите систему и протестируйте её работу с различными запросами
3. Проанализируйте, как система обрабатывает различные типы запросов

## Задание 2: Расширение функциональности
1. Добавьте новые категории рекомендаций (например, музыка, рестораны)
2. Реализуйте обработку ошибок и валидацию входных параметров
3. Добавьте возможность сохранения истории рекомендаций

## Задание 3: Интеграция с внешним API
1. Выберите внешний API для получения реальных данных (например, IMDB для фильмов)
2. Модифицируйте систему для работы с выбранным API
3. Реализуйте кэширование результатов для оптимизации

## Критерии оценки
1. Функциональность:
   - Корректная работа с функциями OpenAI
   - Правильная обработка различных типов запросов
   - Наличие обработки ошибок

2. Качество кода:
   - Читаемость и структурированность
   - Наличие документации
   - Следование PEP 8

3. Дополнительные возможности:
   - Интеграция с внешними API
   - Оптимизация производительности
   - Расширенная функциональность

## Дополнительные задания
1. Добавьте поддержку нескольких языков
2. Создайте простой веб-интерфейс для системы
3. Реализуйте систему рейтинга рекомендаций

## Отчет
При выполнении лабораторной работы необходимо подготовить отчет, содержащий:
1. Описание реализованной функциональности
2. Примеры работы системы
3. Анализ проблем и их решений
4. Предложения по улучшению
