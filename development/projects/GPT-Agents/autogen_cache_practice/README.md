# Практические задания по кэшированию в AutoGen

## Установка

1. Создайте виртуальное окружение:
```bash
python -m venv .venv
source .venv/bin/activate  # для Linux/Mac
# или
.venv\Scripts\activate  # для Windows
```

2. Установите зависимости:
```bash
pip install -r requirements.txt
```

3. Создайте файл `.env` и добавьте ваш API ключ:
```
OPENAI_API_KEY=ваш_ключ_здесь
```

## Задания

### 1. Базовое использование кэша
```bash
python basic_cache.py
```
Этот скрипт демонстрирует базовое использование кэширования с двумя агентами.

### 2. Анализ кэша
```bash
python cache_analysis.py
```
Этот скрипт запускает несколько диалогов с разными значениями `cache_seed` и анализирует размер кэша.

### 3. Оптимизация использования токенов
```bash
python token_optimization.py
```
Этот скрипт сравнивает производительность с кэшированием и без него.

## Примечания
- Перед запуском убедитесь, что у вас есть действующий API ключ OpenAI
- Все скрипты создают папку `.cache` в текущей директории
- Для очистки кэша можно удалить папку `.cache`
