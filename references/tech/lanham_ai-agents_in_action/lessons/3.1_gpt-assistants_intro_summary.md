# Краткий конспект: Раздел 3.1 - Исследование GPT-ассистентов

**GPT-ассистенты (GPTs)** — это настраиваемые версии ChatGPT для специфических задач.

### Ключевые возможности:

1.  **Постоянные инструкции (Custom Instructions):** Задают постоянную "личность" и роль ассистента.
2.  **Расширенные знания (Knowledge):** Позволяют загружать файлы (PDF, TXT), создавая персональную базу знаний.
3.  **Инструменты (Capabilities):**
    - **Code Interpreter:** Выполнение кода Python для анализа данных.
    - **Web Browsing:** Доступ к интернету.
    - **DALL-E:** Генерация изображений.
4.  **Действия (Actions):** Подключение к внешним API для взаимодействия с другими сервисами.

### Создание:

- **Вкладка "Create":** Быстрая настройка через диалог с "GPT Builder".
- **Вкладка "Configure":** Детальная ручная настройка всех параметров (инструкции, знания, инструменты, действия).

### Вывод:

GPT-ассистенты позволяют "упаковать" сложный промпт, дополнительные знания и инструменты в единый, мощный и многоразовый инструмент, превращая LLM в полноценного агента.
